# Fullstack_Vibes_Clinica
 Projeto de desenvolvimento de uma API com operações CRUD para gerenciamento de uma clínica, permitindo o cadastro, consulta, atualização e remoção de dados de pacientes, médicos e atendimentos. A solução visa garantir organização, eficiência e integração das informações, utilizando boas práticas de desenvolvimento e arquitetura de software.
